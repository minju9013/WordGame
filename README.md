# 한글 단어 맞추기 🎈

유아(약 43개월)를 위한 한글 단어 맞추기 웹 게임입니다. 이모지 그림과 단어를 보고 알맞은 카드를 고르거나 선을 이으며 한글을 익혀요.

**배포 버전:** https://minju9013.github.io/WordGame/

## 주요 기능

### 게임 모드
- **🖼️ 그림＋글자**: 이모지와 단어를 함께 보고 같은 단어 찾기
- **🐻 그림만**: 그림을 보고 알맞은 단어 고르기
- **🔤 글자만**: 글자를 읽고 알맞은 그림 찾기
- **🔗 선긋기**: 왼쪽 그림과 오른쪽 글자를 선으로 이어 짝 맞추기 (드래그 또는 점 톡톡)

### 단계(Step) 선택
- 오른쪽 위 **단계 드롭다운**으로 단어 묶음(Step1 / Step2 / Step3 …)을 골라 학습
- 단계마다 서로 다른 단어 세트가 나옵니다

### 피드백 & 효과
- 정답 시 **색종이 폭죽**(canvas-confetti) + **딩동 소리** + **박수 소리** + 칭찬 메시지
- 오답 시 **"띵~" 효과음**과 흔들림 애니메이션으로 부드럽게 다시 시도 유도
- 한국어 **음성 읽어주기**(글자를 못 읽는 아이를 위해)
- 효과음은 외부 파일 없이 **Web Audio**로 생성 (인터넷 없이도 소리 재생)

### 화면 & 사용성
- **이전 / 다음** 문제 이동 (단어 맞추기 모드)
- 대표 **로고 헤더** + 텍스트 탭으로 깔끔한 상단 구성
- 아이패드(특히 **가로 모드**)에 최적화된 큰 버튼·큰 글씨
- 이모지는 **SVG(Twemoji)** 로 렌더링해 기기와 무관하게 동일하게 표시
- 두/세 글자 단어도 카드에 맞게 글자 크기 자동 조정

## 파일 구성

```
index.html   # 화면 구조
styles.css   # 디자인 / 반응형 레이아웃
data.js      # 단계·단어·이모지·메시지 데이터
game.js      # 게임 로직 (단어 맞추기 + 선긋기)
logo.png     # 대표 로고 이미지
docs/        # 상세 문서
```

## 실행 방법

`index.html`을 브라우저로 열면 됩니다. (인터넷 연결 시 폰트·이모지·폭죽 효과가 가장 예쁘게 표시됩니다.)
