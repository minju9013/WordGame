# 단어 / 단계 추가·변경

게임의 단어와 단계는 `data.js`의 `STEPS` 배열만 수정하면 됩니다.

## 기본 구조

```js
const STEPS = [
  {
    id: 1,
    label: "Step1",
    words: [
      { word: "곰", emoji: "🐻" },
      { word: "달", emoji: "🌙" },
      // ...
    ],
  },
  {
    id: 2,
    label: "Step2",
    words: [
      { word: "모자", emoji: "🎩" },
      // ...
    ],
  },
];
```

| 필드 | 설명 |
|------|------|
| `id` | 단계 번호 (고유값) |
| `label` | 드롭다운에 표시되는 이름 (예: `Step1`) |
| `words` | 해당 단계의 단어 목록 |
| `word` | 한글 단어 |
| `emoji` | 그림으로 쓸 이모지 |

## 단어 추가·수정

`words` 배열에 아래 형태로 추가하거나 기존 항목을 수정합니다.

```js
{ word: "문", emoji: "🚪" }
```

## 새 단계(Step) 추가

`STEPS` 배열에 항목을 하나 더 추가하면 **단계 드롭다운에 자동으로 표시**됩니다.

```js
{
  id: 4,
  label: "Step4",
  words: [
    { word: "문", emoji: "🚪" },
    { word: "나무", emoji: "🌳" },
  ],
},
```

## 참고

- **단어 맞추기**: 정답 1개 + 오답 1개를 랜덤으로 출제합니다.
- **선긋기**: 각 단계 단어 중 **3쌍**을 골라 출제합니다.
- 칭찬/격려 메시지는 같은 파일의 `PRAISE`, `ENCOURAGE` 배열에서 수정할 수 있습니다.
